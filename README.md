# 🎵 Mariachis Los Salmos - Bogotá

Sitio web profesional para **Mariachis Los Salmos de Bogotá** - Servicios de música tradicional mexicana para eventos especiales.

## 🌟 Características Principales

- **🎯 Optimizado para SEO**: Meta tags, schema.org, sitemap dinámico
- **📱 Responsive Design**: Adaptado para todos los dispositivos
- **🚀 Rendimiento Optimizado**: Next.js 14 con App Router
- **🎨 Diseño Moderno**: Interfaz transparente y elegante
- **⚡ Velocidad**: Optimización de imágenes con Next.js Image
- **🔍 SEO Avanzado**: Schema.org, robots.txt, sitemap dinámico

## 🛠️ Tecnologías Utilizadas

- **Framework**: Next.js 14 con App Router
- **Estilos**: Tailwind CSS con diseño personalizado
- **Animaciones**: Framer Motion para efectos suaves
- **Optimización**: Next.js Image para imágenes
- **Despliegue**: Vercel y Netlify ready

## 📋 Servicios Ofrecidos

- ✅ **Eventos Privados**: Cumpleaños, aniversarios, fiestas
- ✅ **Bodas**: Ceremonias y recepciones
- ✅ **Serenatas**: Declaraciones de amor y sorpresas
- ✅ **Restaurantes**: Ambientación musical
- ✅ **Eventos Corporativos**: Empresariales y corporativos
- ✅ **Quinceañeras**: Celebraciones tradicionales

## 🚀 Despliegue

### Vercel
```bash
# Configuración automática con Vercel
vercel --prod
```

### Netlify
```bash
# Configuración automática con Netlify
netlify deploy --prod
```

### Desarrollo Local
```bash
npm install
npm run dev
```

## 📁 Estructura del Proyecto

```
my-app/
├── app/                    # App Router de Next.js
│   ├── page.tsx           # Página principal (single page)
│   ├── layout.tsx         # Layout principal con SEO
│   ├── servicios/         # Página de servicios
│   ├── galeria/           # Galería de imágenes
│   ├── repertorio/        # Catálogo de canciones
│   ├── contacto/          # Formulario de contacto
│   └── globals.css        # Estilos globales
├── components/            # Componentes reutilizables
│   ├── Navbar.tsx         # Navegación con scroll suave
│   ├── Footer.tsx         # Pie de página
│   ├── ServiceCard.tsx    # Cards de servicios
│   └── FloatingWhatsApp.tsx # Botón flotante de WhatsApp
├── public/                # Archivos estáticos
│   ├── images/gallery/    # Imágenes optimizadas
│   ├── robots.txt         # Configuración para rastreadores
│   └── site.webmanifest   # PWA manifest
├── vercel.json            # Configuración para Vercel
├── netlify.toml           # Configuración para Netlify
└── README.md             # Este archivo
```
